# DriveNow — Car Rental Management System

DriveNow is a responsive car rental booking website created as part of the **Sqrock IT Solutions Web Development Internship — Phase 1, Task 1**.

## Features

* Home page with modern design
* Browse car fleet
* Search cars by make or model
* Filter cars by category
* View car details
* Book a car using a booking form
* View booking history
* Cancel bookings
* Sign in and sign up using localStorage
* Admin dashboard
* Dark mode
* Responsive design for mobile, tablet, and desktop

## Tech Stack

* React
* Vite
* React Router
* Tailwind CSS
* JavaScript
* lucide-react
* localStorage

## Installation

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```

Preview the build:

```bash
npm run preview
```

## Project Type

This is a frontend-only internship project.
Bookings and user details are stored in the browser using localStorage.
